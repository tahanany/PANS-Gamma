"Object Orianted Programming" <p>

1to10 = Generates a sequence of numbers from 1 to 10 with alternating signs (positive/negative).<p>

leibnitz = Approximates the value of Pi using the Leibniz formula and calculates the deviation from the actual value.<p>

leibnround = Approximates Pi using the Leibniz series and displays the result with specific rounding precision.<p>

leibvseuler = Compares the accuracy of Leibniz and Euler algorithms in approximating the value of Pi.<p>

factorial = Calculates the factorial of a number using both for and while loops.<p>

Fibonacci = Generates the first N terms of the Fibonacci sequence based on user input.<p>

numberguessgame = Implements a "Guess the Number" game with directional feedback (too big/too small) and an attempt counter.<p>

rectan_triangle = Checks if three side lengths form a valid triangle and determines if it is a right-angled triangle.<p>

grad = Manages a list of students with random grades, calculating scholarships and class average using LINQ.<p>

employee = Demonstrates inheritance and polymorphism by managing employees and managers with custom salary raise logic.<p>

student = Utilizes LINQ queries, delegates, and lambda expressions to manage, filter, and analyze student data dynamically.<p>

computer = Simulates a computer network using inheritance to manage different device types and test connectivity via ping.<p>
